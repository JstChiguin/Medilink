# MediLink

Proyecto de arquitectura de microservicios desarrollado con Spring Boot.

## Microservicios
- Agenda Médica
- Cita
- Pago
- Historial Clínico
- Notificación
- Receta Médica
- Usuario
- Videollamada
- Evaluación de Atención

## Tecnologías
- Java 17
- Spring Boot
- Spring Cloud
- Eureka Server
- Spring Cloud Gateway
- OpenFeign
- MySQL
- Docker
- Swagger
- HATEOAS
- Flyway
